# 🛠️ DevOps To-Do Project

A simple and clean **To-Do Web Application** built using **Python (Flask)** with a minimal frontend. This project demonstrates basic DevOps concepts, backend API creation, and file-based task storage.

## 📌 Features
- Add new tasks
- View all tasks
- Remove tasks
- Persistent storage using tasks.txt
- REST API using Flask
- Simple UI

## 🏗️ Project Structure
my-first-project-main/
│
├── app.py
├── app1.py
├── index.html
├── tasks.txt
└── README.md

## ⚙️ Tech Stack
- Backend: Python, Flask  
- Frontend: HTML, CSS  
- Storage: Text File  

## 🚀 How to Run

1. Install Flask:
pip install flask

2. Run:
python app1.py

3. Open:
http://127.0.0.1:5000
